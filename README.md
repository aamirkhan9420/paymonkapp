<h1>User Management</h1>
#  The user Managemen app is a platform which shows the list of users and also enables to edit the user details. it has a feature like pagination and a User Details page.
<h1>Features</h1>
<li>User List </li>
<li>User Details</li>
<li>User Details</li>
<li>User Edit</li>
<li>Pagination</li>
<li>Responsive</li>
